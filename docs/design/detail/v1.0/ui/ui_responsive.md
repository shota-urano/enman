レスポンシブ詳細設計（20250831_v1.0）

ブレークポイント（Tailwind準拠）
- モバイル基準: 360–428px（主要ターゲット）
- sm: 640px, md: 768px, lg: 1024px（将来のレイアウト拡張）

全体方針
- モバイル最適化を最優先。md以上は2カラム（将来）
- セーフエリア対応: env(safe-area-inset-bottom) を考慮して LumaBar 余白を調整
  .safe-bottom{ padding-bottom: max(24px, env(safe-area-inset-bottom)); }

コンテナ/余白
- 余白: px-4（16px）を基本。sm以上で px-6, md以上で px-8
- ヘッダー: sticky, 高さ 56–60px, 背景に透過は使用しない（読みやすさ優先）

ホーム（月カレンダー）
- モバイル: 7列グリッド（grid-cols-7）, セルmin-h-14（56px）
- sm以上: セルに当日バッジと合計チップを増やす
- md以上（将来）: 左に月サマリーカード, 右にカレンダー

取引登録
- モバイル: 1カラム。金額+日付は2列（grid-cols-2 gap-3）
- md以上: 2カラムフォーム（左フォーム、右プレビュー/ヒント）

明細詳細
- モバイル: DialogはSheet（下から）を優先
- md以上: 中央モーダル幅 560px

レポート
- モバイル: タブ切替 + 単一グラフ表示
- sm以上: 2カラムでグラフとテーブルを並列

ナビゲーション（LumaBar）
- 常に中央下固定。キーボード開時は reposition（bottom をキーボード高さに応じて上げる想定）
- スクロール時は隠さない（誤タップ防止のため影を強調）

画像/アイコン
- アイコンは24px基準、retina対応はベクタのみ
- 画像が入る場合は object-cover, rounded-md

タイポグラフィ
- 本文: text-[14px]/[20px]
- メタ情報: text-[12px]/[18px]
- 見出し: text-[16–20px]/[24–28px]
