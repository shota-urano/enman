画面設計・UI仕様（最新UI要件準拠: docs/requirements_ui/20250831_v1.0.md）

前提/ガイドライン
- ターゲット: 女性の主婦。落ち着いた家族感のある配色（柔らかい中間色）。
- トーン: モダン/視認性重視。目がちかちかしないコントラスト設計。
- コンポーネント: shadcn/ui を基本。Tailwind + TypeScript。アイコンは lucide-react。
- 通知: 完了トーストは画面上部から表示。
- ナビ: 提示コンポーネント futuristic-nav（framer-motion）をフッタ固定で活用。

情報設計/主な画面
1) 認証/初回セットアップ
  - サインアップ/サインイン
  - 世帯作成/招待リンク発行/参加
2) ホーム（月カレンダー）
  - 月表示、各日: 収入/支出/差額サマリ
  - 日タップ → 日別明細（モーダル/画面）
3) 取引登録
  - 種別切替（収入/支出）, 金額, 日付, カテゴリ, アカウント, （支出のみ）場所, メモ
  - 3タップ目標, 入力中はローカル下書き自動保存
4) 明細詳細
  - 編集/削除（本人のみ）
  - コメント/スタンプ, 通知連動
5) サブスク
  - 定義一覧/編集
  - 前日21:00の通知 → 金額確定 → 当月分登録
6) レポート（月次サマリー）
  - カテゴリ別円/棒, 合計（収入/支出/差額）, 締め日反映
7) 設定
  - 締め日, カテゴリ/アカウント編集, 招待管理

画面遷移図（概略）
- 認証 → セットアップ → ホーム（月）
- ホーム ←→ 取引登録 / 日明細 / レポート / サブスク / 設定

ワイヤーフレーム（テキスト）
- ヘッダ: タイトル/戻る/検索（将来）
- 本文: カレンダー or フォーム/リスト
- フッタ: ナビ（futuristic-nav）: Home, Search(将来), Alerts(通知), Profile, Saved(将来), Settings

UI/UXガイドライン
- 色: ベース#FAFAFA, アクセント#5B8DEF, サブ#E2E8F0（例）。ダークモードは将来。
- 余白/角丸: shadcn/ui のデフォルトを踏襲
- フィードバック: 成功=上部トースト(2.5s), 失敗=残留/詳細表示
- 入力: 数値はテンキー優先, 日付ピッカーは月カレンダー連動

レスポンシブ対応
- モバイル最適。幅 ≥ 768px では2カラム化（将来）

コンポーネント配置（shadcn準拠）
- ルート: /components/ui にコンポーネントを配置
- futuristic-nav.tsx と demo.tsx を要件通りに配置
  - 依存: lucide-react, framer-motion
  - 利用例: demo.tsx → <LumaBar />

アクセシビリティ
- フォーカスリング/コントラスト確保, アイコンにaria-label, トーストはrole="status"

トレーサビリティ
- コメント/スタンプ/通知: A3
- 3タップ登録/下書き保存: 基本機能/パフォーマンス
- トースト上部表示/ナビ統一: UI要件
